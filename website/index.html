Why TalkyMcTalkface

  • 100% local No data leaves your Mac
  • No subscription Download once, use forever

  • REST API Integrate with your scripts
  • Open source MIT licensed, fork it

Command line ready

$ talky "Your text here" --voice morgan
$ talky --list-voices
$ talky --help

Install the CLI from the app's settings menu.

Requirements

  • macOS 14.0 (Sonoma) or later
  • Apple Silicon recommended
  • ~4GB disk space for TTS model
  • 2-4GB RAM during inference
